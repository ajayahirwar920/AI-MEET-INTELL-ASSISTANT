# MeetMind AI
An AI-powered meeting intelligence platform that converts meeting transcripts and audio into structured, actionable reports.

---------

## ✨ Features

- 🎤 Audio Transcription (MP3, WAV)
- 📄 PDF & TXT Transcript Support
- 🤖 AI Meeting Summarization using Gemini
- ✅ Action Item Extraction
- 🎯 Key Decision Identification
- ⚠️ Risk & Blocker Detection
- 📋 Follow-up Task Generation
- 📊 Meeting Insights Dashboard
- 📥 Export to TXT, PDF, and DOCX

---

## 🛠 Tech Stack

- Python 3.11
- Streamlit
- Google Gemini API
- Faster-Whisper
- python-docx
- ReportLab
- PyPDF
